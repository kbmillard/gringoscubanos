# Gringos Cubanos Logo Web Assets

Included:
- Transparent cropped master PNG
- Responsive PNG logo widths
- WebP logo widths
- Square icon PNGs from 32px to 1024px
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png
- site.webmanifest
- white-background JPG fallback

Recommended usage:
- Header/logo: `logo-width-640.png` or `logo-width-1024.webp`
- Favicon: `favicon.ico`
- Apple/mobile icon: `apple-touch-icon.png`
- PWA icons: `logo-192x192.png` and `logo-512x512.png`
